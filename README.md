# File Sorter

A simple file sorting utility with a graphical interface.

---

## Platform

- Windows 10/11

- ---

## Features

- Sorting files by creation date
- Sorting files by extension
- One-click sorting

---

## Built with

- Python
- Tkinter

---

## How to use

- Select the sorting method in the "Settings"
- Select the folder in which you want to sort by clicking the "Browse" button
- Click "Start sorting"
